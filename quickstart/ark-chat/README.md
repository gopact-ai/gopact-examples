# Ark Chat

Run a single Ark chat completion through `gopact-ext/models/ark`.

```bash
GOPACT_LLM_BASEURL=https://ark.cn-beijing.volces.com \
GOPACT_ARK_REGION=cn-beijing \
GOPACT_ARK_ACCESS_KEY=your-access-key \
GOPACT_ARK_SECRET_KEY=your-secret-key \
GOPACT_LLM_MODEL=ep-20260624181107-glhd6 \
go run ./quickstart/ark-chat
```
